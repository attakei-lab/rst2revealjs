Overview

What is this
------------

This is presentation generated by Reveal.js
using rst2revealjs.

rst2revealjs
------------

rst2revealjs is docutils extension library
to generate Reveal.js HTML from reStructuredText.

Features
--------

* Convert doctree to Reveal.js HTML.
* Generate full HTML using CDN.
* Select color theme (not yet).

Difference from sphinx-revealjs
===============================

sphinx-revealjs
---------------

This is Sphinx extension to generate Reveal.js HTML
from Sphinx documents.

rst2revealjs is not ...
-----------------------

* Not supporting multiple documents.
* Not defined extra directives.
